Un sistema integral de gestión de pedidos para locales gastronómicos, desarrollado bajo la arquitectura **ASP.NET MVC**. Este proyecto simula el flujo completo de una tienda real, desde la gestión de usuarios hasta la facturación final.

## 💎 Características Principales
* **🛒 Experiencia de Compra Completa:** Flujo dinámico que incluye selección de productos, carrito de compras, y elección de método de envío o retiro.
* **👤 Gestión de Usuarios (CRUD):** Sistema de registro e inicio de sesión con administración de perfiles y seguimiento de pedidos.
* **💳 Pasarela de Pago y Entrega:** Integración de lógica para selección de métodos de pago y confirmación de estados de pedido.
* **📑 Facturación Automática:** Generación de tickets de compra tras la confirmación del pedido.
* **🗄️ Persistencia Robusta:** Modelado de datos profesional utilizando **Entity Framework** y **SQL Server**.

## 🛠️ Stack Tecnológico
* **Lenguaje:** C# con .NET MVC (Motor de vistas Razor).
* **ORM:** Entity Framework (Code First/Database First).
* **Base de Datos:** SQL Server.
* **Frontend:** HTML5, CSS3, JavaScript.

## 📂 Estructura del Proyecto
* **Models:** Definición de entidades (Hamburguesas, Bebidas, Usuarios, Pedidos).
* **Views:** Interfaces dinámicas desarrolladas con Razor.
* **Controllers:** Lógica de negocio y manejo de peticiones.
* **Data:** Contexto de base de datos y migraciones de Entity Framework.
